# Waste-Not: Circular Economy Tracker

## Overview
Waste-Not is a circular economy tracking system designed to improve waste segregation and recycling transport efficiency. The project identifies contaminants across different waste collection sites and optimizes truck loads for transport.

## Problem Statement
Many recycling programs fail because recyclable materials get contaminated by non-recyclables. This project analyzes waste materials from different collection sites and detects common contaminants.

## Features
- Identify common waste contaminants using Set operations
- Compare waste materials across multiple collection sites
- Optimize truck loading using Greedy Algorithm (Bin Packing)
- Analyze waste contamination percentage
- Support circular economy initiatives

## Technologies Used
- Python
- Set Data Structures
- Greedy Algorithms
- CSV Data Processing

## Dataset
Waste classification dataset from Kaggle.

## Sample Dataset Structure
site_id | material_list | total_weight_kg | contamination_pct
--- | --- | --- | ---
SITE_NORTH | {PET, HDPE, Glass, Foil} | 450 | 5%

## Goal
The goal is to demonstrate how mathematical approaches like set theory and greedy optimization can improve waste management and recycling systems.
